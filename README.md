# elasticsearch

Prerequisites

1. Install node.js version 5+ from https://nodejs.org

2. Install Oracle JET command line interface

$ npm -g install @oracle/ojet-cli

3. Clone the project

$ git clone https://github.com/oracle/ojet-cli.git

4. Install corsproxy

$ npm install -g corsproxy


Starting the app

1. Start corsproxy

$ corsproxy

2. Start the app

$ ojet serve
